# S01: Local export contract verifier

**Goal:** Create a safe local verification surface for the existing ONNX export artifact, provenance, and manifest contract.
**Demo:** After this, the repo has an executable local verifier for the existing ONNX export contract and a clear statement of what it proves and does not prove.

## Must-Haves

- Verifier validates manifest artifact size/sha, provenance model revision/source files, export metadata toolchain pins, and production_default=false.
- Verifier reports that this is existing-artifact contract verification, not regenerated byte-for-byte proof.
- Diagnostics use safe repo-relative paths.
- Tests/probes cover positive and tampered metadata failure cases.
