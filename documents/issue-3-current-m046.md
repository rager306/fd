- **#6** â€” even the well-batched v1 path peeks cache per-item; each L1 miss synchronously hits Redis, so a cold-L1/warm-L2 batch pays up to 128 sequential Redis GETs. Add a batched `MGetIfPresent` (Redis MGET/pipeline).
